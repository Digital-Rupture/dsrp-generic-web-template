### 2. Directory Structure

This template uses a standardized structure suitable for most modern web applications:

| Directory | Purpose |
| :--- | :--- |
| **`src/`** | Contains all application source code, components, and core logic. |
| **`public/`** | Stores static assets, including the main `index.html` file, images, and fonts. |
| **`docs/`** | Dedicated folder for detailed project documentation, architectural diagrams, and user guides. |
| **`.github/`** | Configuration files for GitHub Actions, CI/CD, and issue templates. |
